# Интернет-магазин

## Описание:

В проекте разрабатывается ядро для интернет-магазина.


## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/ekomendant/E_commerce.git
```

2. Установите зависимости:
```
pip install -r requirements.txt
```


## Модули и примеры использования.

### Модуль main.py
Модуль содержит данные для запуска и проверки работы написанных классов и функций.

### Модуль product.py
Модуль содержит класс `Product` с параметрами продуктов.

### Модуль category.py
Модуль содержит класс `Category` с параметрами категории продуктов (в том числе содержат в себе перечень продуктов).

### Модуль product_iterator.py
Модуль содержит класс `ProductIterator` для поочередного вывода продуктов одной категории.

### Модуль utils.py
Модуль содержит 2 функции:
1. `read_json` - функция принимает на вход путь до JSON-файла и возвращает список словарей с данными о категориях и 
продуктах.
2. `create_objects_from_json` - функция принимает список словарей и преобразовывает данные в объекты классов.


## Тестирование:

Тесты всех функций реализованы в директории `tests`. 
Для запуска тестов введите в терминале команду:
```
pytest
```

Для формирования отчета о покрытии тестами введите в терминале команду:
```
pytest --cov=src --cov-report=html
```
Результат покрытия тестами в HTML-формате можно посмотреть в [отчете](htmlcov/index.html).
